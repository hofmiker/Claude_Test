# Toy Box Rescue — 2D-Fangspiel im Kinderzimmer

## Live-URL
https://hofmiker.github.io/Claude_Test/toy-box-rescue/

## Dateien
- `index.html` — komplettes Spiel (Canvas/Vanilla-JS, keine externen Assets)

## Features
- Woody (Emoji-Sprite) fängt herabfallende Spielzeuge (Stern, Ballon,
  Puzzleteil, Auto, Dino) und weicht Zurgs Robotern aus
- 3 Leben, Punktezähler, steigende Fallgeschwindigkeit/Spawnrate je Level
- Start- und Game-Over-Overlay mit Neustart-Button
- Tastatur- (Pfeiltasten/A-D) und Touch-Steuerung (On-Screen-Buttons,
  automatisch sichtbar auf Touch-Geräten)

## Steuerung
Pfeiltasten links/rechts oder A/D bewegen Woody. Auf Touch-Geräten
erscheinen ◀️/▶️-Buttons unten am Bildschirmrand.

## Tech-Stack
- Reines Canvas 2D + Vanilla JS, kein Build-Schritt, keine Abhängigkeiten
- Emoji statt lizenzierter Grafiken, um keine Marken-/Urheberrechte
  von Disney/Pixar zu verletzen (siehe Hinweis im Footer der Seite)
