# Spice Wars — 2D Sci-Fi Echtzeitstrategie (Dune-2-Stil)

## Live-URL
https://hofmiker.github.io/Claude_Test/spice-wars/

## Dateien
- `index.html` — komplettes Spiel (Canvas/Vanilla-JS, keine externen Assets)

## Features
- Basisbau (Kommandozentrale, Raffinerie, Kaserne, Fabrik, Geschützturm)
- Spice-Ressourcenwirtschaft mit Harvestern
- Echtzeitkampf mit A*-Pathfinding, Fog of War, einfacher KI-Gegner
- Steuerung: Links-Klick auswählen/Rahmen ziehen, Rechts-Klick bewegen/angreifen/ernten

## Tech-Stack
- Reines Canvas 2D + Vanilla JS, kein Build-Schritt, keine Abhängigkeiten
